# Using Status Report

## Dashboard

The dashboard shows all your projects at a glance:

- **Progress bars** show overall completion and checkpoint progress
- **Status badges** indicate project health (on-track, at-risk, off-track, completed, paused)
- **Next checkpoint** shows upcoming milestones with dates
- Click any project card to view its details

## Managing Projects

### Create a Project

1. Click "New Project" on the dashboard
2. Fill in the project name, description, and initial status
3. Set the first checkpoint date and description
4. Add team members and tags

### Project Detail View

The project detail page shows:

- **Task List**: All tasks with status, assignee, priority, and due date
- **Status Timeline**: History of all status updates with task changes
- **Progress**: Visual progress indicators

### Task Management

- **Add tasks**: Click "Add Task" to create new work items
- **Update status**: Click a task's status badge to cycle through todo, in-progress, blocked, done
- **Reorder**: Drag tasks to change priority order
- **Edit**: Click a task to edit its details

## AI-Powered Status Updates

### Quick Status Update

1. Navigate to a project and click "Update Status"
2. The AI agent starts a conversation asking about your progress
3. Answer questions about:
   - What work was completed since last update
   - Any blockers or risks
   - What's planned next
4. The agent automatically:
   - Updates task statuses based on your answers
   - Records a status update with markdown summary
   - Suggests project status changes if needed (e.g., at-risk)

### Cross-Project Chat

Use the general chat to ask questions across all projects:

- "Which projects are at risk?"
- "What are the current blockers?"
- "Summarize progress on Project X"
- "What was completed last week?"

The status-assistant agent searches across all projects and updates to provide answers.

## Project Statuses

| Status | Meaning |
|---|---|
| **On Track** | Project is progressing as planned |
| **At Risk** | Some concerns but recoverable |
| **Off Track** | Significant issues requiring attention |
| **Completed** | All work finished |
| **Paused** | Work temporarily stopped |
